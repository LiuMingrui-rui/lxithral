网站结构与技术栈

2026.04.18 · 技术分享 · 约 3 分钟阅读

探索 L'xithral Journal 的技术架构与设计理念。

项目结构

lxithral/
├── index.html          # 主页
├── about.html          # 关于页面
├── style.css           # 全局样式
├── script.js           # 交互逻辑
├── posts/              # 文章目录
│   ├── post-1.html     # 第一篇文章
│   └── post-2.html     # 第二篇文章
└── res/                # 资源文件
    ├── tou.jpg         # 头像图片
    └── MiSansVF.ttf    # 可变字体

核心技术

HTML5 - 语义化标签构建清晰的文档结构，使用 <header>、<main>、<article> 等元素提升可访问性。

CSS3 - 现代 CSS 特性实现丰富的视觉效果：

  • 玻璃拟态效果（backdrop-filter）
  • CSS Grid 和 Flexbox 响应式布局
  • CSS 变量实现主题切换
  • 关键帧动画和过渡效果

JavaScript (ES6+) - 原生 JS 实现所有交互功能，无需框架依赖：

  • Intersection Observer API 滚动显示动画
  • localStorage 持久化用户偏好
  • requestAnimationFrame 优化滚动性能
  • 事件委托和防抖处理

特色功能

可变字体支持 - 集成 MiSans VF 可变字体，通过 font-variation-settings 实现实时字重调节（100-900），用户可以根据喜好自定义阅读体验。

双主题系统 - 深色/浅色模式无缝切换，自动检测系统偏好并保存用户选择。使用 CSS 变量统一管理颜色方案，确保一致性。

动态背景 - 多层背景效果结合：点状移动背景、渐变流光动画、模糊光斑，营造沉浸式的视觉氛围。

3D 头像交互 - 鼠标跟随的 3D 倾斜效果，使用 CSS transform 和 perspective 实现真实的透视变换。

简洁而不简单，优雅且高性能。

设计理念

L'xithral Journal 坚持「零依赖」原则——不使用任何框架或库，完全依靠原生 Web 技术。这不仅减少了加载时间，更让我能够深入理解每一个效果的实现原理。

所有动画都经过精心调优，使用 cubic-bezier 缓动函数创造自然的运动曲线。性能方面，通过 will-change、transform 和 opacity 属性触发 GPU 加速，确保流畅的 60fps 体验。

这是一个持续进化的项目，每一行代码都承载着对 Web 技术的热爱与探索。
