动态规划入门：背包问题

2026.04.19 · C++竞赛 · 约 6 分钟阅读

背包问题是动态规划的经典题型，在信息学竞赛中频繁出现。掌握这类问题的解法对提升算法能力至关重要。

0/1背包问题

有N件物品和一个容量为V的背包。第i件物品的费用是c[i]，价值是w[i]。求解将哪些物品装入背包可使价值总和最大。

#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1005;
int dp[MAXN];  // dp[j]表示容量为j时的最大价值

int main() {
    int N, V;
    cin >> N >> V;
    
    for (int i = 0; i < N; i++) {
        int c, w;  // 费用和价值
        cin >> c >> w;
        
        // 逆序遍历，确保每件物品只选一次
        for (int j = V; j >= c; j--) {
            dp[j] = max(dp[j], dp[j - c] + w);
        }
    }
    
    cout << dp[V] << endl;
    
    return 0;
}

完全背包问题

与0/1背包不同，完全背包中每种物品都有无限件可用。只需将内层循环改为正序即可。

#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1005;
int dp[MAXN];

int main() {
    int N, V;
    cin >> N >> V;
    
    for (int i = 0; i < N; i++) {
        int c, w;
        cin >> c >> w;
        
        // 正序遍历，允许重复选择
        for (int j = c; j <= V; j++) {
            dp[j] = max(dp[j], dp[j - c] + w);
        }
    }
    
    cout << dp[V] << endl;
    
    return 0;
}

多重背包问题

每种物品有有限件num[i]。可以使用二进制优化将其转化为0/1背包问题。

#include <bits/stdc++.h>
using namespace std;

const int MAXN = 10005;
int dp[MAXN];

struct Item {
    int cost, value;
};

int main() {
    int N, V;
    cin >> N >> V;
    
    vector<Item> items;
    
    // 二进制拆分
    for (int i = 0; i < N; i++) {
        int c, w, num;
        cin >> c >> w >> num;
        
        int k = 1;
        while (k <= num) {
            items.push_back({c * k, w * k});
            num -= k;
            k *= 2;
        }
        if (num > 0) {
            items.push_back({c * num, w * num});
        }
    }
    
    // 0/1背包求解
    for (auto& item : items) {
        for (int j = V; j >= item.cost; j--) {
            dp[j] = max(dp[j], dp[j - item.cost] + item.value);
        }
    }
    
    cout << dp[V] << endl;
    
    return 0;
}

解题技巧总结

  • 状态定义：明确dp数组的含义，通常是dp[i][j]或dp[j]
  • 状态转移：找出子问题之间的关系，写出转移方程
  • 边界条件：初始化dp数组，处理特殊情况
  • 空间优化：使用滚动数组将二维降为一维

动态规划的核心是「记忆化搜索」，避免重复计算子问题。多做题、多总结，才能熟练掌握这一重要思想。

建议从简单的背包问题开始练习，逐步过渡到更复杂的DP题目，如最长公共子序列、编辑距离等经典问题。
