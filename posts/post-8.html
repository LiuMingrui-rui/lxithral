澎湃OS 2~3.3 设置修改教程

2026.04.19 · ROM定制 · 约 8 分钟阅读

本教程来自 @kakashi1v1，酷安 @rkpsz。授权转载并修改部分代码，如果你在ROM中使用了本设置修改教程，需要标注原作者名字。

重要声明：无论付费还是免费使用，都需要标注 kakash1 或 rkpsz 的名字。

准备工作

首先需要反编译设置APK。你可以：

  • 使用西米露开启包管理设置里面的全部内容，然后直接安装
  • 或者修改内置进ROM包

提示：想改文字可以直接在 miui_version_card 中修改，卡片第一行和第三行都可以自定义。长按版本卡片可以切换风格。

修改 DeviceBasicInfoPresenter

搜索类：com/android/settings/device/DeviceBasicInfoPresenter

替换以下方法：

.method private getLineNum()I
    .registers 3

    iget-boolean v0, p0, Lcom/android/settings/device/DeviceBasicInfoPresenter;->isUseMiui15CardStyle:Z

    const/4 v1, 0x2

    if-eqz v0, :cond_6

    return v1

    :cond_6
    iget-object p0, p0, Lcom/android/settings/device/DeviceBasicInfoPresenter;->mContext:Landroid/content/Context;

    invoke-static {p0}, Lcom/android/settings/MiuiUtils;->isLandScape(Landroid/content/Context;)Z

    move-result p0

    if-eqz p0, :cond_15

    invoke-static {}, Lcom/android/settings/utils/SettingsFeatures;->isSplitTabletDevice()Z

    move-result p0

    if-eqz p0, :cond_15

    const/4 v1, 0x3

    :cond_15
    return v1
.end method

隐藏保修期和本机权益

使用 MT 管理器反编译设置，搜索方法：

.method public static isLocalCnAndChinese()Z

替换方法内容为：

.registers 2

const/4 v0, 0x0

return v0
.end method

导入 Smali 文件

在 classes10.dex 中导入和替换 smail.zip 里面的全部类文件。

修改 ARSC 资源

如果你有会员：使用 arsc编辑器++ 打开 resources.arsc 添加资源

如果没有会员：使用普通arsc编辑器，手动一个一个添加（参考视频教程）

可选修改

更改 my_device_info_item_background_color 为 #55ffffff

这样 light mode 也有模糊效果（可选，可能导致其它卡片有点不好看）

添加 Drawable 资源

----drawables------

add in drawables:

    <path name="ic_device_oos_inner">res/drawable/ic_device_oos_inner.xml</path>
    <path name="ic_device_oos_bg">res/drawable/ic_device_oos_bg.xml</path>
    <path name="oos_progress_drawable">res/drawable/oos_progress_drawable.xml</path>

add in type-info for drawable:

    <entry id="0x7f085e90" name="ic_device_oos_inner" />
    <entry id="0x7f085e91" name="ic_device_oos_bg" />
    <entry id="0x7f085e93" name="oos_progress_drawable" />

添加 ID 资源

------ids-------

add in ids:

    <id name="storage_progress_kchi" />
    <id name="device_name_in_banner" />
    <id name="build_dev_info" />
    <id name="device_info_wallpaper_blur_bg" />
    <id name="about_phone_rom_name_info" />

add in type-info for ids:

    <entry id="0x7f0b5e51" name="storage_progress_kchi" />
    <entry id="0x7f0b5e52" name="device_name_in_banner" />
    <entry id="0x7f0b5e53" name="build_dev_info" />
    <entry id="0x7f0b5e54" name="device_info_wallpaper_blur_bg" />
    <entry id="0x7f0b5e55" name="about_phone_rom_name_info" />

添加 Layout 资源

------layout--------

add in layout:

    <path name="device_info_item_kashi">res/layout/device_info_item_kashi.xml</path>
    <path name="storage_info_item_kashi">res/layout/storage_info_item_kashi.xml</path>

add in type-info for layout:

    <entry id="0x7f0e566e" name="device_info_item_kashi" />
    <entry id="0x7f0e566f" name="storage_info_item_kashi" />

如果你没有会员就手动添加，教程看视频。

添加布局文件

在 /res/layout 添加两个layout，复制对应txt文件内容进去。另外两个文件 device_layout.xml 和 miui_version_card.xml 直接复制txt内容进行替换。

V2 修复（不要替换整个 Settings）

找到 /res/layout/my_device_settings.xml

定位到以下内容：

                <View
                    android:id="@id/version_card_click_view"
                    android:layout_width="match_parent"
                    android:layout_height="wrap_content" />

将 height 改为 178dp：

                <View
                    android:id="@id/version_card_click_view"
                    android:layout_width="match_parent"
                    android:layout_height="178dp" />

Drawable 替换

Drawable 也是一样的，只用替换就行。视频教程中一开始没修改drawable里面的内容，后面发现需要修改。

注意事项：所有修改前请务必备份原文件，修改完成后重新编译APK并测试功能是否正常。

完成以上步骤后，你的澎湃OS设置页面就会呈现出全新的样式和布局。记得在使用过程中标注原作者信息！
