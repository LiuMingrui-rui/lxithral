C++快速排序算法详解

2026.04.19 · C++竞赛 · 约 5 分钟阅读

快速排序是信息学竞赛中最常用的排序算法之一，掌握它对提高解题效率至关重要。

算法原理

快速排序采用分治策略，通过选择一个基准元素（pivot），将数组分为两部分：左边小于基准，右边大于基准，然后递归地对两部分进行排序。

基础实现

#include <bits/stdc++.h>
using namespace std;

// 快速排序核心函数
void quickSort(vector<int>& arr, int left, int right) {
    if (left >= right) return;
    
    int pivot = arr[left];  // 选择第一个元素作为基准
    int i = left, j = right;
    
    while (i < j) {
        // 从右向左找第一个小于pivot的元素
        while (i < j && arr[j] >= pivot) j--;
        arr[i] = arr[j];
        
        // 从左向右找第一个大于pivot的元素
        while (i < j && arr[i] <= pivot) i++;
        arr[j] = arr[i];
    }
    
    arr[i] = pivot;  // 放置基准元素
    
    // 递归排序左右两部分
    quickSort(arr, left, i - 1);
    quickSort(arr, i + 1, right);
}

int main() {
    vector<int> arr = {38, 27, 43, 3, 9, 82, 10};
    
    quickSort(arr, 0, arr.size() - 1);
    
    for (int num : arr) {
        cout << num << " ";
    }
    
    return 0;
}

优化版本

在竞赛中，我们通常会使用一些优化技巧来提高快速排序的性能：

  • 三数取中法选择pivot，避免最坏情况
  • 小数组使用插入排序
  • 尾递归优化减少栈空间
#include <bits/stdc++.h>
using namespace std;

// 三数取中法选择pivot
int medianOfThree(vector<int>& arr, int left, int right) {
    int mid = left + (right - left) / 2;
    
    if (arr[left] > arr[mid])
        swap(arr[left], arr[mid]);
    if (arr[left] > arr[right])
        swap(arr[left], arr[right]);
    if (arr[mid] > arr[right])
        swap(arr[mid], arr[right]);
    
    // 将中位数放到right-1位置
    swap(arr[mid], arr[right - 1]);
    return arr[right - 1];
}

// 优化的快速排序
void optimizedQuickSort(vector<int>& arr, int left, int right) {
    // 小数组使用插入排序
    if (right - left < 10) {
        for (int i = left + 1; i <= right; i++) {
            int key = arr[i];
            int j = i - 1;
            while (j >= left && arr[j] > key) {
                arr[j + 1] = arr[j];
                j--;
            }
            arr[j + 1] = key;
        }
        return;
    }
    
    int pivot = medianOfThree(arr, left, right);
    int i = left, j = right - 1;
    
    while (true) {
        while (arr[++i] < pivot) {}
        while (arr[--j] > pivot) {}
        if (i < j)
            swap(arr[i], arr[j]);
        else
            break;
    }
    
    swap(arr[i], arr[right - 1]);
    
    optimizedQuickSort(arr, left, i - 1);
    optimizedQuickSort(arr, i + 1, right);
}

时间复杂度分析

最好情况：O(n log n) - 每次都能均匀分割数组

平均情况：O(n log n) - 随机数据下的期望复杂度

最坏情况：O(n²) - 数组已有序且每次都选到最大或最小值

在实际竞赛中，建议使用STL的sort函数，它结合了快速排序、堆排序和插入排序的优点，性能更加稳定。

理解快速排序的原理对于解决更复杂的分治问题非常有帮助，建议多练习相关题目来加深理解。
